# Roadmap

## Current Status
Working MVP foundation with frontend, backend, shared schemas, fixtures, deploy/smoke scripts, and tests in the original private working project. This public showcase repo focuses on safe reviewability rather than full operational completeness.

## Milestone 1: Discovery / Intake Hardening
- normal brief normalization;
- JTBD-style intake notes;
- bounded multi-pass discovery;
- deterministic scoring and filtering;
- richer grounding metadata.

## Milestone 2: Extraction / Normalization
- selected-URL extraction as a stable path;
- normalization layer;
- citation-ready metadata;
- partial failure handling.

## Milestone 3: Pattern / Strategy Pipeline
- deduplication;
- stable cluster formation;
- overused angle generation;
- opportunity generation;
- next-test brief generation;
- evidence-vs-inference discipline.

## Milestone 4: Interleaved Story Layer
- story layer inside `BoardPayload`;
- audio recap;
- optional generated-image story block.

## Milestone 5: Reliability / Demo Packaging
- low-source path;
- audio-unavailable path;
- judge access;
- hosted deployment proof;
- recorded happy path;
- testing instructions;
- budget and rate-limit guards.
