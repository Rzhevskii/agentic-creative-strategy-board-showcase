# Showcase Scope

## What this public repository includes
- public-facing product framing;
- architecture and roadmap docs;
- shared JSON schemas;
- sanitized example requests and payloads;
- sanitized demo fixtures;
- selected backend and frontend code excerpts;
- publication audit and checklist materials.

## What it intentionally excludes
- secrets and local env files;
- logs and operational traces;
- internal planning workspaces;
- private deployment runbooks;
- full cloud wiring;
- unsafe or unnecessary infrastructure details.

## Why a separate repo exists
The original private working repository contains real engineering progress but also includes materials that are not safe or useful for a public showcase. This sibling repo exists so reviewers can evaluate the project honestly without exposing internal or sensitive artifacts.
